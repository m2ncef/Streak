## Installation
1. Clone the repository

```
git clone https://github.com/m2ncef/Streak.git
```

2. Navigate to the project folder

```
cd Streak
```

3. Install dependencies
   
```
npm install
```

4. Running Locally
   
```
npm start
```
   
Open http://localhost:3000 in your browser to view the app.

### Build
```
npm run build
```
